# TownyAdvanced 汉化配置文件

本项目旨在将 Minecraft 插件 Towny Advanced 的配置文件汉化为中文。

## 进度

下面列出了每个配置文件的汉化进度：

+ [ ] config.yml
- [ ] database.yml
- [ ] townyperms.yml
- [ ] zh-CN.yml

## 如何贡献

我们欢迎您为本项目做出贡献！请遵循以下步骤：

1. Fork 本仓库。
2. 在本地克隆您的 fork。
3. 在相应的文件中进行翻译或改进。
4. 提交 Pull Request，描述您的更改以便我们审阅。

## 如何使用

1. 下载最新的 Towny Advanced 插件。
2. 将zh-CN.yml放入对应的插件目录中。
3. 在config.yml中将 language 设置成中文。
4. 重新启动服务器以应用更改。
